Configuración inicial

Antes de usar la app debes conectarla
con tu Google Sheet. Sigue los pasos:

Paso 1 — Abrir Apps Script

Abre tu Google Sheet ↗ y ve a Extensiones → Apps Script

Paso 2 — Pegar el código

Borra el contenido existente y pega el código del archivo apps-script.gs que viene junto a esta app. Guarda con Ctrl+S.

Paso 3 — Desplegar como Web App

  1. Clic en Implementar → Nueva implementación
  2. Tipo: Aplicación web
  3. Ejecutar como: Yo (mi cuenta)
  4. Quién tiene acceso: Cualquier persona
  5. Clic en Implementar y autoriza los permisos
  6. Copia la URL del servicio web que aparece
